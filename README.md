# GameLife | RPG 风格人生管理应用

[![Platform](https://img.shields.io/badge/Platform-Xiaomi%20Watch%20(Vela%20JS)-blue)](https://iot.mi.com/vela/quickapp)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**GameLife** 是一款专为小米手环9 Pro（Vela JS 平台）打造的 RPG 风格人生管理应用。它将枯燥的日常任务转化为有趣的 RPG 冒险，通过经验值、金币、属性点和成就系统，激励你完成现实生活中的目标。

---

## 🌟 核心理念

**“把生活当作一场游戏，你就是主角。”**

GameLife 不仅仅是一个待办事项清单，它是一个完整的个人成长引擎。通过在手表端随时随地的交互，让自我管理变得像刷副本一样令人上瘾。

---

## ✨ 功能亮点

### ⚔️ 委托系统 (Quest System)
*   **多样化任务**：支持普通点击、计数模式（如：喝 8 杯水）和计时模式（专注时长）。
*   **灵活频率**：单次、每日、每周、每月、每 N 天、无限次，满足所有习惯养成需求。
*   **智能排序**：根据紧急程度、重要性和截止日期自动调整优先级，确保你始终专注于最重要的事。
*   **连击奖励**：连续完成周期性任务可获得额外经验和金币加成。

### 🎭 角色成长 (RPG Stats)
*   **属性矩阵**：通过不同类型的任务提升**力量**、**智力**、**魅力**，甚至支持自定义属性。
*   **升级系统**：积累经验值 (Exp) 提升等级，解锁更高阶的成就。
*   **成长日志**：详尽记录你的每一次进步，回顾成长的点点滴滴。

### 🏪 集市与仓库 (Shop & Inventory)
*   **自定义奖励**：在商店中创建属于你自己的奖励（如：玩 1 小时游戏、喝杯奶茶）。
*   **金币经济**：完成任务赚取金币，在商店中“购买”奖励。
*   **抽奖与盲盒**：支持概率抽奖模式，为生活增添惊喜。

### ⏳ 时光沙漏 (Tomato Timer)
*   **番茄钟集成**：专注模式与任务系统深度绑定，完成专注即可自动推进委托进度。
*   **全方位配置**：自定义专注/休息时长，支持自动开始和专注奖励。

### 🏦 中央银行 (Bank System)
*   **存款与利息**：将金币存入银行赚取日息。
*   **贷款机制**：急需金币？可以向银行贷款，但要记得及时还款以避免利息堆积。

### 🏆 激励与责任 (Accountability)
*   **成就系统**：数十种内置成就，记录你从“初出茅庐”到“行动大师”的旅程。
*   **惩罚机制**：任务逾期将面临经验值或金币扣除，保持紧迫感（可自定义比例）。

## 🚀 快速上手

### 1. 开发环境
确保已安装 [aiot-toolkit](https://www.npmjs.com/package/aiot-toolkit)。

```bash
npm install
```

### 2. 运行与构建
*   **启动开发服务器**：
    ```bash
    npm run start
    ```
*   **构建生产包**：
    ```bash
    npm run build
    npm run release
    ```
